Memento

给你的 AI 工具建个共享大脑 — 采集、搜索、跨设备。一条命令装完整栈。
macOS · Linux
curl -fsSL https://mem.ihasy.com/install.sh | sh
Windows (PowerShell)
iwr https://mem.ihasy.com/install.ps1 -useb | iex
需要 Docker Desktop / Engine, Python 3.11+, 6GB 空余内存。脚本会自动下载仓库、生成随机密钥、起 容器、建首个用户、装采集器。

自定义

  • MEMENTO_INSTALL_DIR — 安装目录 (默认 ~/memento)
  • MEMENTO_VERSION — git ref (默认 main)
  • 完成后:./install.sh embedding 装语义搜索服务; ./install.sh doctor 看状态

手动方式

  • GitHub 仓库 — 直接 clone
  • API 文档 (Swagger)
